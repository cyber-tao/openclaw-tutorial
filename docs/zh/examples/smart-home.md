# 智能家居实战场景

使用 OpenClaw 轻松掌控你的智能家居。

## 概览

OpenClaw 可以与你的智能家居系统深度集成：
- 🏠 Home Assistant 联动
- 📹 监控摄像头实时查看
- 💡 灯光与氛围控制
- 🌡️ 环境与气候调节
- 📱 跨平台设备通知

## 初始设置

### 安装智能家居相关技能

```bash
# 摄像头集成
openclaw skill install camsnap

# 桌面/UI 自动化
openclaw skill install peekaboo
```

### 配置 Home Assistant

```json
{
  "smarthome": {
    "homeAssistant": {
      "url": "http://homeassistant.local:8123",
      "token": "${HA_TOKEN}"
    }
  }
}
```

## 自动化思路

### 早间例程

```bash
openclaw cron add --name "morning-routine" \
  --schedule "0 7 * * 1-5" \
  --task "逐步调亮灯光，开启咖啡机，播报今日天气"
```

### 晚间安防

```bash
openclaw cron add --name "evening-security" \
  --schedule "0 22 * * *" \
  --task "检查门锁，开启安防模式，关闭非必要灯光"
```

### 节能优化

```bash
openclaw cron add --name "energy-check" \
  --schedule "0 */4 * * *" \
  --task "检查空房间是否有灯未关，并提醒关闭"
```

## 专家建议

1. **测试自动化逻辑** - 在正式依赖之前多次验证
2. **设置手动覆盖** - 确保在系统故障时可以手动控制
3. **定期维护** - 检查感应器电池和连接状态
4. **隐私保护** - 尽可能在局域网内处理数据

务必使用强身份验证保护你的智能家居集成接口！
:::
